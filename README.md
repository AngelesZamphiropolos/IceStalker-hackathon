# IceStalker---hackathon

Género: Terror / Suspenso, exploración
Vista: top-down
Inspiración temática: Penguin Academy


Descripción del juego

Juego de terror y exploración ambientado en Penguin Academy. Tras una falla en el generador eléctrico durante la noche, el protagonista queda atrapado dentro del edificio.

El objetivo del jugador es recolectar fusibles para restaurar la energía y activar el ascensor, único medio de escape. Para conseguirlos, debe explorar el entorno y completar dos minijuegos integrados al mapa.

Durante la exploración, un monstruo controlado por IA recorre el edificio. Si el jugador es capturado tres veces, pierde la partida.

Condición de victoria: restaurar la energía y escapar en el ascensor.
Condición de derrota: ser atrapado tres veces por el enemigo.
